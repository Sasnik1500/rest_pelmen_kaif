# rest_pelmen_kaif



# Пельмени с кайфом — заказ еды и управление товарами

Десктопное приложение на JavaFX для просмотра каталога товаров, оформления заказов и управления системой.  
Позволяет пользователям выбирать товары, добавлять их в корзину и оформлять заказы, а администраторам — управлять товарами, пользователями, заказами и промокодами.

## Возможности

- **Каталог товаров** — просмотр списка товаров с категориями и описанием.
- **Поиск товаров** — поиск по названию, категории и описанию.
- **Корзина** — добавление товаров, изменение количества, удаление из корзины.
- **Оформление заказа** — создание заказа на основе выбранных товаров.
- **Авторизация и регистрация** — вход в систему и создание нового аккаунта.
- **Личный кабинет** — просмотр заказов пользователя.
- **Админ-панель** — управление товарами, пользователями, заказами, промокодами и логами.
- **Фильтрация товаров** — отображение товаров по категориям.
- **Работа с Supabase** — хранение и получение данных через REST API.

## Структура проекта

- **controllers** — контроллеры JavaFX и логика интерфейса.
- **models** — модели данных (`User`, `Product`, `Order`, `CartItem`).
- **services** — сервисы для работы с данными и Supabase.
- **utils** — вспомогательные классы (`Session`, `SceneUtils`).
- **resources** — FXML-файлы интерфейса и ресурсы приложения.

## Основные модули

### Пользовательская часть
- регистрация и вход;
- просмотр каталога;
- поиск и фильтрация товаров;
- добавление товаров в корзину;
- оформление заказа.

### Административная часть
- просмотр и управление пользователями;
- добавление, редактирование и удаление товаров;
- скрытие и показ товаров;
- просмотр заказов;
- управление промокодами;
- просмотр логов действий.
